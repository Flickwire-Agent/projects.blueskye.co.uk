# projects.blueskye.co.uk

React landing page for `projects.blueskye.co.uk` — the project portal and agentic infrastructure dashboard.

## What it does

- Lists all discovered projects (auto-detected from `~/dev/projects/`)
- Shows live health status for each project (up / degraded / unreachable)
- Displays OpenCode token usage by provider/model for the last day, week, and lifetime
- Uses **Mantine** component library with a dark theme

## Stack

| Layer | Tech |
|-------|------|
| Framework | React 19 + TypeScript |
| Bundler | Vite 8 |
| UI | Mantine 9 + Tabler Icons |
| Data | Static JSON (`projects.json`, `health.json`, `opencode-token-usage.json`) generated by local scripts |

## Development

```bash
pnpm dev        # dev server with HMR
pnpm build      # typecheck + build + regenerate project/health JSON
pnpm preview    # serve the production build locally
```

## Deployment

After `pnpm build`, the compiled `dist/` is served statically by Caddy. The `health.json` is regenerated by the `scan-projects` systemd timer every 5 minutes. OpenCode token usage is refreshed by `~/bin/opencode-token-usage` during builds.
